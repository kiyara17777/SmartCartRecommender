# Smart Cart Recommender

A C++ console-based shopping cart system that allows users to log in, browse products, add items to a cart, and receive smart product recommendations based on their selections.

---

## Features

- User login and registration
- Browse products with prices
- Add items to cart
- View cart and total cost
- Get product recommendations
- File-based data handling

---

## File Structure

SmartCartRecommender/

├── main.cpp          // Application source code  
├── users.txt         // Stored user credentials  
├── products.txt      // Product names  
├── prices.txt        // Product prices  
├── rules.txt         // Recommendation rules  
├── cart.txt          // Cart data  


---

## How to Run

1. Clone the repository:

git clone https://github.com/your-username/SmartCartRecommender.git

cd SmartCartRecommender

2. Compile the program:

g++ main.cpp -o SmartCart

3. Run the program:
   
./SmartCart
